# Genie AR Button

WebComponent to directly launch the AR view of Genie AR models.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Build <ar-button> WebComponent
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
